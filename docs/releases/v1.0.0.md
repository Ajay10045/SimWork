# SimWork v1.0.0

Release date: 2026-03-17

## Summary

SimWork `v1.0.0` is the first production release of the PM simulation platform. This release packages the scenario-driven investigation workflow, role-based AI teammate experience, scoring flow, authentication work, and the minimum CI and deployment configuration required to ship the product on Vercel and Railway.

## Highlights

- Production-ready frontend and backend deployment configuration for Vercel and Railway.
- PR-gated CI covering backend lint and tests plus frontend lint and production build.
- Scenario investigation flow with evidence capture, final submission, and review/scoring surfaces.
- Cleanup of legacy frontend and backend files that conflicted with the current app-router architecture.
- Railway health check aligned to the live backend `/health` endpoint.

## Operational notes

- Frontend production branch: `main`
- Backend production branch: `main`
- Backend health endpoint: `/health`
- Required environment variables remain managed in deployment dashboards, not in git.

## Verification

- Frontend production build passed.
